Smart Mirror

Objective

Smart-Mirror is basically an LCD panel sandwiched between a observational mirror that allows act as a one-way mirror and a wooden frame that displays real-time information. The whole system was powered by raspberry pi 3.
Our goal was to develop the user-interface from scratch and to add some extra functionalities to it in order to improve the efficiency.

Functionalities added

  • The user interface was developed using tkinter(GUI) library in python.
  • Ultrasonic sensor was used detect objects in front of the mirror and toggle the screen on/off.
  • Pulse-rate sensor was used detect pulse-rate and display it on the mirror.
  • LDR sensor was used to automatically switch on/off lights.
  • Some extra information like top 3 emails, weather info, news feed were added using API's.
